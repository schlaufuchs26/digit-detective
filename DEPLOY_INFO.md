## 🎮 How to Play

1. Visit: https://schlaufuchs26.github.io/digit-detective/
2. Click "Start Game"
3. Read the sequence with missing digits (blanks)
4. Type the missing digits
5. Press Submit or Enter
6. See instant feedback
7. Continue for 25 rounds
8. View final stats and category breakdown

## 💾 Data Persistence

- Last 10 game scores stored in browser localStorage
- Score data includes:
  - Final score
  - Accuracy percentage
  - Best streak
  - Timestamp

Access in browser console: `JSON.parse(localStorage.getItem('digitDetectiveScores'))`
